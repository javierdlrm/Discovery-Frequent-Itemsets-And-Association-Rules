    \hypertarget{introduction}{%
\section{Introduction}\label{introduction}}

This document is a report created as an assignment for ID2222 course at
KTH. The aim of the assignment was to implement the stages of finding
textually similar documents based on Jaccard similarity using the
shingling, minhashing, and locality-sensitive hashing (LSH) techniques
and corresponding algorithms. We used text abstracts from NSF Research
Award Abstracts 1990-2003 Data Set {[}1{]} to test our solution. The
example set of documents is saved in the repository.

    \hypertarget{solution}{%
\section{Solution}\label{solution}}

We decided to develop a project in Python 3.7 using popular libraries
pro-vided in \texttt{requirements.txt} file in the repository. The whole
repository itself is attached to this report. In order to fulfill the
task assigned we developed two classes: 1. Document, representing a
piece of text and providing methods for text transformation 2.
Comparator being a wrapper for collecting the data, creating
aforementioned Document classes and generating project results.

    \hypertarget{running}{%
\section{Running}\label{running}}

In order for the solution to run, Python 3.7 has to be installed. We
recommendcreating a virtual environment for the purpose of evaluating
the solution. One can either use command line \texttt{python\ run.py} to
save graphs to file or run the programme in attached jupyter notebook:
\texttt{example.ipynb}. Below is the output of the jupyter notebook
